## 스포츠업종 분석(서울)
### 실내운동시설과 실외운동시설이 많은 지역 알아보기
  - 실외보다 실내 운동시설 수가 훨씬 많았다.
  - 강남구가 실내, 실외 운동시설 모두 다른 구보다 많았다.(Why?)
  - 강남구를 제외하고는 실내,실외 모두 서초구 송파구 순서로 많았고 그 이후로는 실내는 강서구, 강동구 순서였고, 실외는 영등포구, 관악구 순서였다.
  - 실내, 실외 운동시설에 어떤 종류가 있는지 알아보는게 좋을 것 같다.
